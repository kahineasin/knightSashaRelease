Game Introduction

  • This game is about the battle between beautiful girls and monsters

Game Features

  1. Beautiful girl characters, multiple characters, and male characters as well
  2. Supports multiple gamepads such as Xbox, Switch, and PS5, with all button mappings unified and represented using PS5 button layouts. Keyboard operation is also supported
  3. KOF-style battles, multiplayer melee, the ability to add computer AI characters, and the inclusion of a "frame bar" similar to that in Street Fighter VI
  4. Increase character attributes infinitely
  5. Various timing pressure mechanisms can train your reaction. For example, in KOF mode, you can immediately defend when Sasha strikes. If you press L1 within 0.15 seconds, you can block it
  6. The catching skill, after being caught, you must input according to the instructions above the character's head to break free
  7. There are passionate plots
  8. It can flexibly set key mappings for gamepads, keyboards, and even mobile phones, and also supports multi-key combinations
  9. The mobile app also supports game controllers
  10. The developer promises permanent updates, including adding new characters, enhancing graphics and materials, improving game mechanics, and fixing any bugs that may arise

The operation method of the game

  1. The character can defend, dodge, double jump, and run
  2. Skill combo commands, such as the finishing move which is ↓↘→↓↘→□, are intentionally strict by the developer. They may be harder to execute than in KOF, but they can better exercise your button precision
  3. The Sasha character can simplify their moves. For example, pressing the directional keys + △ will activate a skill, while pressing △ + ○ will activate a finishing move
  4. Sasha's normal attack consists of a multi-branch combo, such as pkpk, pkkp, kpk

游戏安装说明

  • pc端如果没有jdk环境:
    1. 下载 PC端exe版本, 解压后作为游戏主目录. (以后版本更新时, 不需要这一步)
    2. 下载 jar, 把jar移动到 knightSasha_pc_exe/lib/ 目录下.
    3. 运行 knightsasha.exe
  • pc端已有jdk环境: 只需要下载 jar版本, 在相同目录下执行cmd指令: java -jar -"Dfile.encoding=UTF8" -"Dsun.jnu.encoding=UTF8" knightsasha.jar
  • 手机端: 直接下载 apk 安装即可

(附)各品牌的手柄匹配系统的方法:

  • PS5手柄
    • android手机: 按住手柄上的截图键和PS键, 直到灯闪(进入匹配蓝牙状态)
    • pc端: 方法同android端
  • SWITCH PRO 手柄
    • android手机: 手柄需用数据线连接电源(或插电脑上), 按住手柄上的线插口旁边的小圆钮(不可松开), 灯闪匹配蓝牙
    • pc端: 方法同android端
