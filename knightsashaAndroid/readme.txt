# 游戏名称
KnightSasha(安卓平台, 安卓电视平台)

# 游戏运行
1. 直接安装knightsasha.apk即可

# 游戏玩法
本游戏在安卓平台上同样支持所有游戏手柄（xbox one, switch, PS5)，连接多个手柄即可多人游玩

## 单人游玩
从“进入游戏”菜单进入，一直按手柄的A键即可开始游戏

## 本地多人对战
从“进入格斗游戏模式”进入即可

## 本地多人合作
从“进入3d多人合作模式”进入即可

# 角色操作
"手柄->键盘"键位的对应关系(支持改键位):
x->z, y->s, a->x, b->c, R1->v, R2->f

## 视野操作
远:L1+R2  近:L1+L2  重置视野:L1+L2+R2

## Sasha
1. 普通连招 x->y->x->y 或 y->x->y 或 x->y->y->x
2. 跳:a, 躲闪:→b, 防御:b, →→跑, 飞行:空中按住R2
3. 气功 R1 或 ↓↘→x
4. 升剑 →R1 或 →↘↓↘→x
5. 突进剑 R1+y 或 ↓↘→↓↘→x
6. 被怪物捉住时，要按血量条旁边的组合键位提示来解除

## 巨人哥布林
1. 普通攻击 x
2. 抱人 ↓↘→↓↘→x
