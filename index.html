游戏介绍

  • 本游戏是美少女与怪物之间的战斗

游戏特色

  1. 美少女角色，多角色，也有男角色
  2. 支持xbox switch PS5 等多种手柄, 所有手柄键位统一，以PS5按键表示, 也支持键盘操作.
  3. 可kof式对战, 多人混战, 可加入电脑ai角色
  4. 无限提高人物属性
  5. 多种目压机制可以训练你的反应, 比如kof模式时, 你可以在sasha剑击时立刻防御, 你在0.15秒内按下L1, 就能防住
  6. 捕捉技能, 被捉住后, 要按人物头上指令来输入, 才能挣脱
  7. 有激情剧情
  8. 能很灵活地设置 手柄 键盘 甚至手机 的键位, 还支持多键组合
  9. 手机端也支持游戏手柄
  10. 开发者承诺永久更新，包括增加角色, 素材美化更新, 游戏机制提高, 如果有bug必定修复

游戏的操作方式

  1. 人物可防御, 闪避, 二段跳, 奔跑.
  2. 技能搓招, 比如必杀技是↓↘→↓↘→□, 开发者特意使搓招指令严格, 也许比kof更难搓, 但更能锻炼你的按键精准度.
  3. sasha角色可以简化出招, 比如 方向键+△ 可以出技能, △+○ 可以出必杀
  4. sasha角色的普通攻击是多分支的连续技, 比如pkpk, pkkp, kpk

游戏安装说明

  • pc端如果没有jdk环境:
    1. 下载 PC端exe版本, 解压后作为游戏主目录. (以后版本更新时, 不需要这一步)
    2. 下载 jar, 把jar移动到 knightSasha_pc_exe/lib/ 目录下.
    3. 运行 knightsasha.exe
  • pc端已有jdk环境: 只需要下载 jar版本, 在相同目录下执行cmd指令: java -jar -"Dfile.encoding=UTF8" -"Dsun.jnu.encoding=UTF8" knightsasha.jar
  • 手机端: 直接下载 apk 安装即可

(附)各品牌的手柄匹配系统的方法:

  • PS5手柄
    • android手机: 按住手柄上的截图键和PS键, 直到灯闪(进入匹配蓝牙状态)
    • pc端: 方法同android端
  • SWITCH PRO 手柄
    • android手机: 手柄需用数据线连接电源(或插电脑上), 按住手柄上的线插口旁边的小圆钮(不可松开), 灯闪匹配蓝牙
    • pc端: 方法同android端
